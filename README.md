# Pagina de perdon

## Probar localmente

1. Ejecuta `python perdon.py`
2. Abre `http://localhost:8000`

## Personalizar

- Cambia los textos en `index.html`
- Reemplaza las imagenes por fotos tuyas y de tu pareja
- Cambia el enlace de la cancion en la seccion "Nuestra cancion"

## Compartir por link

Puedes subir estos archivos a GitHub Pages, Netlify o Vercel para obtener un enlace publico.
